В данном разделе описывается процесс разворота сайта МКК "с нуля".
Для разворота сайта предлагается использовать VPS-сервер арендованный где-либо под управлением ОС Ubuntu Server.
Инструкция писалась для Ubuntu Server 18.04. Для других дистрибутивов Linux не проверялось.

Разворот состоит из нескольких этапов:
1. Настройка доступа к серверу через SSH с использованием ключа (сертификата)
2. Установка системных требований для сайта (с помощью скрипта)
3. Собственно разворот сайта (с помощью скрипта)
4. Восстановление базы данных из дампа и конвертация данных на новую версию
5. Проверка работы сайта, исправление настроек, и устранение непредвиденных проблем
6. Введение сервера в работу на основной домен сайта
